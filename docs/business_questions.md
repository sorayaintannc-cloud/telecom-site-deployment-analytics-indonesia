# Business Questions and Decision Mapping

| ID | Business Question | Primary Stakeholder | Decision Supported | KPI / Output |
|---|---|---|---|---|
| BQ-01 | How many sites are planned, active, completed, overdue, and at risk? | Project Director, Project Manager | Assess portfolio health | Total Sites, Active Sites, Completion Rate, High-Risk Sites |
| BQ-02 | What percentage of completed sites were finished on time? | Project Manager | Evaluate schedule reliability | On-Time Completion Rate |
| BQ-03 | Which milestones are the main bottlenecks? | Project Manager, Project Controller | Prioritize intervention | Overdue milestones, delay days, SLA compliance |
| BQ-04 | Which issue categories contribute most to delay? | Project Manager, Regional Manager | Target recurring causes | Aging, resolution days, Pareto |
| BQ-05 | Which vendors have strong or weak SLA performance? | Vendor Management | Correct performance and balance capacity | Vendor SLA, backlog, completion rate |
| BQ-06 | Which regions or deployment types have the highest backlog? | Regional Manager | Allocate support | Backlog, completion rate, variance |
| BQ-07 | Which technically complete sites are blocked by documents? | Document Control | Prioritize recovery | Completeness, rejection, revision |
| BQ-08 | Which sites should be prioritized this week? | Project Manager | Create escalation list | Risk score, issue aging, staleness |
| BQ-09 | Is progress improving or deteriorating? | Project Director | Evaluate trajectory | Planned vs actual trend |
| BQ-10 | Are data-quality failures distorting KPIs? | Data/BI Team | Protect reporting trust | DQ pass rate, freshness, rejected rows |

Every dashboard visual must answer at least one documented business question.
